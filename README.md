## Getting Started

Web frontend run: 
* Install flutter plugin via Intelij UI
* Install Flutter - https://docs.flutter.dev/get-started/install
* Then:
    ```shell
    flutter doctor
    flutter pub get
    ```
* Then:
    ```shell
    flutter run -d chrome
    ```

Mock backend run:  
* Run the ServerMockApplication.java
